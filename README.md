# Jupyter with iDg Kernel

## Install & Run

### Get it from source

Get this repository.

### Build it

Build it with `docker build -t sckn/jupyter-dg:latest`.

### Run it

Run it with `docker run -it --rm -p 8888:8888 sckn/jupyter-dg:latest`.
